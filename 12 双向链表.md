# 双向链表

> 优点：无论给出哪个节点，都能遍历链表
>
> 缺点：多耗费一个引用空间，构建双向链表比较复杂

```js
class DoubleLinkedList {
  constructor(val) {
    this.val = val;
    this.next = null;
    this.prev = null;
  } 
  
}

const n1 = new DoubleLinkedList(1);
const n2 = new DoubleLinkedList(2);
const n3 = new DoubleLinkedList(3);

n1.next = n2;
n2.prev = n1;
n2.next = n3;
n3.prev = n2;

console.log(n1);
```
