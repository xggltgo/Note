# 栈和队列

> 用JS实现一个栈

```js
class Stack {
  _arr = [];

  push(val) {
    this._arr.push(val);
  }
  pop(val) {
    this._arr.pop();
  }
}

const stack = new Stack();
stack.push(1);
stack.push(2);
stack.push(3);
console.log(stack._arr);
stack.pop();
console.log(stack._arr);
```

> 用JS实现一个队列

```js
class Queue {
  _arr = [];

  push(val) {
    this._arr.push(val);
  }

  pop() {
    this._arr.shift();
  }
}

const queue = new Queue();
queue.push(1);
queue.push(2);
queue.push(3);
console.log(queue._arr);
queue.pop();
console.log(queue._arr);
```
